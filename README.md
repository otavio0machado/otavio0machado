## 🏆 Projeto destaque

### [BioQC — Lab Quality Control Platform](https://github.com/otavio0machado/bioqc) `Java · Spring Boot · React · TypeScript · PostgreSQL`

Plataforma de Controle de Qualidade para laboratórios clínicos. Versão sanitizada de um sistema em produção, publicada com autorização do cliente.

- **406** testes backend passando · **15** migrations Flyway · **30+** entidades JPA
- Motor de regras Westgard (1-2s, 1-3s, 2-2s, R-4s, 4-1s, 10x)
- PDF com cadeia de assinaturas hash-encadeada (tamper-evident) + verificação por QR code
- Rastreabilidade de reagentes ISO 15189 · RBAC + audit log · JWT com rotação
- Comentário gerado por IA (Gemini) opcional · Métricas Prometheus

🔗 **[Demo ao vivo](https://bioqc.vercel.app)** · **[Case study](https://github.com/otavio0machado/bioqc/blob/main/CASE_STUDY.md)** · **[Landing](https://otavio0machado.github.io/bioqc/)**
